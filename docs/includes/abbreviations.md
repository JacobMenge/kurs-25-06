*[API]: Application Programming Interface – Schnittstelle zur Kommunikation zwischen Softwarekomponenten
*[REST]: Representational State Transfer – Architekturstil für Web-APIs
*[HTTP]: Hypertext Transfer Protocol – Protokoll für die Datenübertragung im Web
*[HTTPS]: HTTP Secure – Verschlüsselte Variante von HTTP
*[CRUD]: Create, Read, Update, Delete – Die vier grundlegenden Datenbankoperationen
*[SQL]: Structured Query Language – Abfragesprache für relationale Datenbanken
*[NoSQL]: Not Only SQL – Datenbanken, die nicht nur auf SQL basieren
*[OOP]: Objektorientierte Programmierung – Programmierparadigma mit Klassen und Objekten
*[CLI]: Command-Line Interface – Kommandozeilenschnittstelle
*[GUI]: Graphical User Interface – Grafische Benutzeroberfläche
*[IDE]: Integrated Development Environment – Integrierte Entwicklungsumgebung
*[JWT]: JSON Web Token – Standard für sichere Authentifizierung
*[SSH]: Secure Shell – Verschlüsseltes Netzwerkprotokoll
*[DNS]: Domain Name System – System zur Auflösung von Domainnamen
*[HTML]: Hypertext Markup Language – Auszeichnungssprache für Webseiten
*[CSS]: Cascading Style Sheets – Stylesheet-Sprache für Webdesign
*[DOM]: Document Object Model – Programmierschnittstelle für HTML-Dokumente
*[JSX]: JavaScript XML – Syntaxerweiterung für React-Komponenten
*[JSON]: JavaScript Object Notation – Leichtgewichtiges Datenaustauschformat
*[YAML]: YAML Ain't Markup Language – Daten-Serialisierungsformat
*[AWS]: Amazon Web Services – Cloud-Computing-Plattform
*[EC2]: Elastic Compute Cloud – Virtuelle Server bei AWS
*[S3]: Simple Storage Service – Objektspeicher bei AWS
*[npm]: Node Package Manager – Paketmanager für JavaScript
*[CORS]: Cross-Origin Resource Sharing – Sicherheitsmechanismus im Browser
*[URL]: Uniform Resource Locator – Webadresse
*[URI]: Uniform Resource Identifier – Eindeutige Ressourcenkennung
*[VM]: Virtuelle Maschine – Emulierte Computerumgebung
*[RAM]: Random Access Memory – Arbeitsspeicher
*[CPU]: Central Processing Unit – Prozessor
*[SSD]: Solid State Drive – Flash-Speicher-Festplatte
*[HDD]: Hard Disk Drive – Magnetische Festplatte
*[BIOS]: Basic Input/Output System – Firmware zur Hardware-Initialisierung
*[UEFI]: Unified Extensible Firmware Interface – Moderner BIOS-Nachfolger
*[LPI]: Linux Professional Institute – Zertifizierungsorganisation für Linux
*[VENV]: Virtual Environment – Isolierte Python-Umgebung
*[ORM]: Object-Relational Mapping – Abbildung von Objekten auf Datenbanktabellen
*[SMTP]: Simple Mail Transfer Protocol – Protokoll zum E-Mail-Versand
*[TCP]: Transmission Control Protocol – Verbindungsorientiertes Netzwerkprotokoll
*[UDP]: User Datagram Protocol – Verbindungsloses Netzwerkprotokoll
*[IPv4]: Internet Protocol Version 4 – Netzwerkprotokoll mit 32-Bit-Adressen
*[IPv6]: Internet Protocol Version 6 – Netzwerkprotokoll mit 128-Bit-Adressen
*[DHCP]: Dynamic Host Configuration Protocol – Automatische IP-Adresszuweisung
*[CIDR]: Classless Inter-Domain Routing – Methode zur IP-Adressierung
*[VPC]: Virtual Private Cloud – Isoliertes virtuelles Netzwerk in der Cloud
*[IAM]: Identity and Access Management – Zugriffsverwaltung bei AWS
*[CI/CD]: Continuous Integration / Continuous Deployment – Automatisierte Software-Auslieferung
*[DevOps]: Development + Operations – Kombination von Entwicklung und IT-Betrieb
*[SPA]: Single Page Application – Webanwendung auf einer einzigen Seite
*[SSR]: Server-Side Rendering – Serverseitige HTML-Generierung
*[CSR]: Client-Side Rendering – Clientseitige HTML-Generierung
